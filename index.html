External Dependencies¶

For Configure the Base we use a set of external ansible roles, hosted at Ansible Galaxy and listed in requirements.yml

---
- name: arillso.logrotate
- name: geerlingguy.docker
- name: geerlingguy.pip
- name: linuxhq.elrepo
- name: paulfantom.restic
- name: robertdebock.bootstrap
- name: robertdebock.epel
- name: robertdebock.fail2ban
- name: ypsman.sshd_config
- name: manala.timezone

Your can use the ansible-galaxy command for installation,

ansible-galaxy install -r requirements.yml
